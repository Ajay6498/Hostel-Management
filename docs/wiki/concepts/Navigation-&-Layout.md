# Navigation & Layout

- Modern top navigation with brand on the left, page links, search, and user menu.
- Mobile view uses a hamburger to toggle a compact menu.
- Role-based visibility: links appear only if your role can access the page.

Key files:
- `src/components/Layout.jsx` – nav bar and responsive mobile menu
- `src/styles.css` – `.topbar`, `.topnav`, `.hamburger`, `.mobile-menu`
